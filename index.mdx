# Welcome to Wovly

Wovly is your **autonomous communication agent** - a privacy-focused AI assistant that manages your inbox, replies, and follow-ups across all your communication platforms.

<Warning>
  **Beta Notice:** Wovly is currently in beta on macOS. Windows and Linux support coming soon. We appreciate your feedback! [Report bugs or request features](https://github.com/wovly/wovly/issues) on GitHub.
</Warning>

## Features

<CardGroup cols={2}>
  <Card title="Multi-Platform Communication" icon="comments">
    **Unified Inbox** – Manage Email, Slack, iMessage, WhatsApp, Telegram, and Discord from one interface
    
    **Voice Mimic** – Learns your communication style per contact and platform
    
    **Cross-Platform** – "Text my wife, then Slack my boss" in a single command
  </Card>
  <Card title="Autonomous Task Execution" icon="robot">
    **Natural Language Tasks** – Describe your goals: "monitor my email and schedule appointments"
    
    **Persistent Follow-ups** – Monitors for replies and follows up intelligently
    
    **Smart Scheduling** – Automatically books meetings via calendar or Calendly
  </Card>
  <Card title="Intelligent Memory" icon="brain">
    **Long-term Memory** – Remembers facts, relationships, and preferences
    
    **Daily Logs** – Automatic conversation summarization and context retention
    
    **Personal Profile** – Knows your family, job, location, allergies, preferences
  </Card>
  <Card title="Customizable Skills" icon="wand-magic-sparkles">
    **Teach Once, Automate Forever** – Define SOPs that Wovly follows consistently
    
    **Personal Playbooks** – Create reusable workflows tailored to your role
    
    **Guardrails** – Set constraints like "never auto-send to executives"
  </Card>
  <Card title="Remote Interfaces" icon="mobile">
    **Talk to Wovly Anywhere** – Chat via WhatsApp or Telegram from your phone
    
    **Full Capability Access** – Run tasks, check emails, send messages remotely
    
    **Real-time Notifications** – Get task alerts and updates wherever you are
  </Card>
  <Card title="Browser Automation" icon="globe">
    **Web Research** – Navigate websites, extract data, fill forms
    
    **Credential Management** – Securely stored logins for automated authentication
    
    **Anti-Detection** – Built-in measures to reduce bot detection
  </Card>
  <Card title="Privacy-First" icon="lock">
    **Local Storage** – App data stays on your machine
    
    **LLM API Calls** – Chat sent to your chosen provider via API
    
    **No Wovly Servers** – Direct API calls, nothing passes through us
  </Card>
</CardGroup>

## Getting Started

<Steps>
  <Step title="Install Wovly">
    Clone the repository and run `npm install` from the root directory
  </Step>
  <Step title="Configure LLM Provider">
    Add your API key from Anthropic, OpenAI, or Google in Settings
  </Step>
  <Step title="Connect Platforms">
    Link your communication platforms from the Integrations page
  </Step>
  <Step title="Start Chatting">
    Ask Wovly to help with emails, scheduling, or research
  </Step>
</Steps>
